# Blender GOH GEM Exporter v0.8.0-pre1

This prerelease packages the current Blender-side GOH authoring workflow into a GitHub-ready layout.

## Focus Areas

- Blender-native GOH presets and helper panels
- Basis metadata support
- SOEdit-friendly round-trip defaults
- Legacy MultiScript text compatibility
- Primitive collision support for `Box`, `Sphere`, and `Cylinder`

## Recommended Audience

- GOH modders moving from 3ds Max to Blender
- Users who still need partial compatibility with old MultiScript property buffers
- Authors who want SOEdit validation without going through 3ds Max

## Included In This Prerelease

- Addon source in `blender_goh_gem_exporter/`
- Regression tests in `tests/`
- Installation and quick-start docs in `docs/`
- `README`, `CHANGELOG`, `CONTRIBUTING`, and `LICENSE`

## Current Status

This is a prerelease packaging pass, not a final marketplace-style distribution.
It is suitable for GitHub publishing, local testing, and manual addon installation from zip.
